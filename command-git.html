Daftar Command Git Bash

  • git init : untuk me-inisialisasi-kan folder yang akan di aktifkan git nya
  • git add . : untuk meng add file yg baru / file yg masih unstage menjadi stage, tanda titik untuk semua file
  • git commmit -m "" : untuk meng-commit dengan -m untuk message di dalam tanda "", kalau yg di commit file yg ter modifikasi maka pakai -am
  • git branch : untuk melihat branch
  • git branch ... : untuk membuat branch
  • git merge ... : untuk merge dengan branch ... , kalau mau merge pastikan pointer aktif pada main/master
  • git branch --merged : untuk melihat branch mana yg sudah ter merge
  • git branch -d ... : untuk men-delete branch, gunakan -D untuk mendelete branch yg belum ter merge
  • git checkout ... : untuk pindah antara branch atau peindah ke commit tertentu
  • git log : untuk melihat log commit seluruhnya
  • git config --global user.name "" : untukmengubah user name dengan config global
  • git config --global user.email "" : untuk mengubah email user dengan config global, pilih local untuk local atau folder saat ini
  • git status : untuk melihat status, ini paling penting
  • git log --all --decorate --oneline --graph : untuk melihat graph
  • alias graph="git log --all --decorate --oneline --graph" : untuk membuat alias dalam satu session yg berisi command git log ...
